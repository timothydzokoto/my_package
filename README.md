# my_learning_package

Return the top `n` numeric values from a list.

## Installation

```bash
pip install my_learning_package
```

## Usage

```python
from my_learning_package import top_n

print(top_n([8, 3, 2, 7, 4], 3))  # [8, 7, 4]
```

## API

`top_n(items, n)` -> list

- `items`: list of numeric values
- `n`: number of highest values to return
- returns: top `n` values in descending order
